# Magic

这是一个构建且可以编译 magic 项目的 node.js 包.

作者: SQJM

许可: GPL-3.0



## 使用方法



#### 下载 magic 

下载完整源码




#### 环境搭建

##### windows

打开系统环境变量

在 Paths 添加 "...path\\magic\\"



#### 新建项目

```cmd
magic init
```

然后依次输入信息



#### 构建项目

```cmd
magic build /YourProjectDir
```




#### 运行项目

```cmd
magic build /YourProjectDir
```
